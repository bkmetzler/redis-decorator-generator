# Creating Releases

## Create a pull request to 'main' branch and merge
## Fetch all existing tags
```git tag --list```

## List all existing tags
```git tag <VERSION>```

VERSION should be a standard 3 versions (0.0.0)
```git tag 0.0.1```

## Push tag up
```git push origin <VERSION>```

Example

```git push origin 0.0.1```
